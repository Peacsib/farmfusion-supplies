# FarmFusion Supplies - Premium Agricultural Website

A modern, responsive website for FarmFusion Supplies, Zimbabwe's trusted agricultural supplier offering quality feeds, fertilizers, and veterinary products.

## 🌟 Features

### Design & UI
- **Premium Scroll Animations** - Powered by GSAP ScrollTrigger for smooth, professional interactions
- **Responsive Design** - Optimized for all devices (desktop, tablet, mobile)
- **Farm-Themed Side Menu** - Custom mobile navigation with agricultural icons
- **Modern Layout** - Clean bento grid design with trust badges
- **Accessibility Compliant** - ARIA labels, keyboard navigation, and screen reader support

### Animations & Interactions
- Smooth scroll-triggered animations
- Parallax effects on hero sections
- Staggered card animations
- Magnetic button effects
- Counter animations
- Text reveal animations
- Image scale effects
- Hover lift interactions

### Pages
1. **Home** - Hero section, featured products, trust badges
2. **Products** - Complete product catalog with filtering
3. **About** - Company information, mission, vision
4. **Contact** - Contact form, branch locations, map integration

### Technical Features
- GSAP 3.12.5 for premium animations
- ScrollTrigger plugin for scroll-based effects
- Intersection Observer API fallback
- Optimized performance with GPU acceleration
- Reduced motion support for accessibility
- SEO optimized with meta tags
- Progressive Web App ready

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/yourusername/farmfusion-supplies.git
cd farmfusion-supplies
```

2. Open in browser:
```bash
# Simply open index.html in your browser
# Or use a local server:
python -m http.server 8000
# Then visit http://localhost:8000
```

## 🔧 Development

### Adding New Animations

Add animation classes to HTML elements:
```html
<div class="fade-up">Content</div>
<div class="scale-in hover-lift">Card</div>
```

### Customizing Colors

Edit CSS variables in `style.css`:
```css
:root {
    --primary-green: #2E7D32;
    --dark-green: #1B5E20;
    /* ... */
}
```
